# Portfolio site

Ένα έτοιμο, στατικό portfolio για GitHub Pages.

## Πριν το ανεβάσεις

Άλλαξε στο `index.html`:

- `Alex Morgan` με το ονοματεπώνυμό σου.
- `hello@yourname.com` με το email σου.
- Τα τέσσερα `https://example.com` με links προς τα πραγματικά projects σου.
- Τίτλους, περιγραφές και το κείμενο About.

## Ανέβασμα στο GitHub Pages

1. Δημιούργησε ένα νέο GitHub repository, π.χ. `portfolio`.
2. Ανέβασε **τα αρχεία μέσα σε αυτόν τον φάκελο** (`index.html`, `style.css`, `README.md`) στη ρίζα του repository.
3. Στο GitHub: **Settings → Pages → Build and deployment → Deploy from a branch**.
4. Διάλεξε `main` branch και `/ (root)`, μετά πάτησε **Save**.
5. Σε λίγα λεπτά θα εμφανιστεί το public URL, συνήθως `https://USERNAME.github.io/portfolio/`.

Δεν χρειάζεται server ή εγκατάσταση πακέτων — είναι απλό HTML/CSS.
